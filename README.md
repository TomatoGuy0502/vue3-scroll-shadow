# Vue 3 Scroll Shadow

Add UX-friendly shadow to your scrollable list

## Installation

```bash
npm i vue3-scroll-shadow
```

## Usage

```javascript
import VueScrollShadow from 'vue3-scroll-shadow'
```

```html
<template>
  <vue-scroll-shadow>
    <!-- Your content goes here -->
  </vue-scroll-shadow>
</template>
```
